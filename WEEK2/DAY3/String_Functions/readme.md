# 📚 Topics Covered

## 🔹 1. Date Functions
- Extracting date parts (YEAR, MONTH, DAY, DAYNAME)
- Finding differences between dates
- Formatting and converting date values
- Working with CURRENT_DATE and CURRENT_TIMESTAMP
- Filtering records using date conditions and ranges

---

## 🔹 2. NULL Handling Functions
- Using `IS NULL` and `IS NOT NULL`
- Replacing missing values using `IFNULL()` and `COALESCE()`
- Conditional NULL handling using `NULLIF()`
- Managing incomplete salary, bonus, and transaction data
- Data cleaning for analytical processing

---

## 🔹 3. Numeric Functions
- Rounding numbers using `ROUND()`
- Ceiling and floor operations (`CEIL()`, `FLOOR()`)
- Truncation using `TRUNCATE()`
- Mathematical operations (`MOD()`, `POWER()`, `SQRT()`, `ABS()`)
- Value comparisons using `GREATEST()` and `LEAST()`
- Formatting numeric outputs using `FORMAT()`

---

## 🔹 4. String Functions
- Case conversion using `UPPER()` and `LOWER()`
- Extracting substrings using `SUBSTRING()`, `LEFT()`, `RIGHT()`
- Combining strings using `CONCAT()` and `CONCAT_WS()`
- Searching and replacing text using `INSTR()`, `LOCATE()`, `REPLACE()`
- Data cleaning and formatting techniques

---

## 🔹 5. Real-World SQL Applications
- Salary calculation and adjustments
- Discount and payment computations
- Handling missing and incomplete records
- Data transformation for reporting systems
- Analytical query building using multiple SQL functions

---

## 🔹 6. Multi-Function Query Building
- Combining STRING + NUMBER + NULL functions
- Nested function transformations in SELECT queries
- Creating derived analytical columns
- Preparing datasets for dashboards and reporting
